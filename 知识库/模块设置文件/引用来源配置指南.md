# 引用来源设置指南

## 概述
在就医决策助手中，**引用来源显示**是建立患者信任的关键功能。本指南说明如何配置引用来源标签和免责声明。

---

## 一、开启引用来源显示

### 扣子平台配置
1. 进入Bot的「知识库」设置
2. 找到「显示引用来源」选项
3. ✅ **开启** 该选项

### 本项目配置
已在API路由 `src/app/api/chat/route.ts` 中配置：

```typescript
// 引用来源设置规范
const CITATION_SETTINGS = `
### 📎 引用来源设置规范
【格式规范、标注要求...】
`;
```

---

## 二、来源标签设置

### 来源类型与格式

| 来源类型 | 标签格式 | 示例 |
|---------|---------|------|
| CSCO指南 | 【CSCO指南2025】 | 【CSCO指南2025】肝转移灶R0切除是唯一潜在治愈手段 |
| NCCN指南 | 【NCCN指南2026】 | 【NCCN指南2026】推荐贝伐单抗用于晚期肠癌 |
| 专家共识 | 【专家共识】 | 【专家共识】转化治疗窗口期2-6个月 |
| 医院官方 | 【来源：XX医院官网】 | 【来源：北京大学肿瘤医院官网】 |
| 政府机构 | 【来源：国家卫健委】 | 【来源：国家卫健委】癌症防治核心信息 |
| 医保局 | 【来源：国家医保局】 | 【来源：国家医保局】异地就医备案政策 |
| 知识库-专家 | 【知识库：熊猫群】 | 【知识库：熊猫群】推荐专家信息 |
| 知识库-保险 | 【知识库：保险】 | 【知识库：保险】惠民保投保要点 |

### 引用优先级
1. **指南/共识类**：CSCO指南 > NCCN指南 > 专家共识
2. **官方类**：政府官网 > 医院官网 > 权威平台
3. **知识库**：专家库 > 指南库 > 保险库

---

## 三、引用标注规则

### 必须标注的情况

| 情况 | 规则 | 示例 |
|------|------|------|
| 数据/统计 | 必须标注 | 【CSCO指南2025】完整切除者5年生存率>40% |
| 治疗方案 | 必须标注 | 【NCCN指南2026】推荐FOLFOXIRI方案 |
| 医院/专家 | 必须标注 | 【知识库：熊猫群】推荐XX医院XX教授 |
| 保险内容 | 必须标注 | 【知识库：保险】惠民保既往症可报销 |
| 费用/报销 | 必须标注 | 【来源：国家医保局】异地就医报销政策 |

### 可选标注的情况
- 通用医学知识：可标注「综合医学文献」

## 四、免责声明设置

### 必选项

回答末尾必须包含以下免责声明：

```
---

**📋 信息来源声明**
本回答参考了以下来源：
• 【CSCO指南2025】2025 CSCO结直肠癌诊疗指南
• 【知识库：熊猫群】肿瘤专家信息汇总
• 【来源：国家卫健委】国家卫生健康委员会官网

**⚠️ 重要提示**
以上信息仅供参考，不能替代专业医生的诊断和治疗。
如有不适，请尽快就医；具体治疗方案请遵医嘱。
```

### 免责声明要点

| 要点 | 说明 |
|------|------|
| 仅供参考 | 明确说明信息仅供参考 |
| 不替代诊疗 | 强调不能替代专业医生诊断 |
| 及时就医 | 提醒如有不适尽快就医 |
| 遵医嘱 | 强调具体方案请遵医嘱 |

## 五、完整回答示例

### 用户问：肝转移还能手术吗？

### Bot回答：

```
【能，但需评估】手术是治愈的最佳机会，但需要满足条件。

【通俗解释】
能手术切除，但前提是转移灶数量少、能切干净、身体状况允许。就像"割韭菜"，要把所有转移灶一次性切干净才有可能治愈。

【专业依据】
【CSCO指南2025】肝转移灶R0切除是唯一潜在治愈手段
【文献数据】完整切除者5年生存率40-50%，未切除者<10%
【知识库：熊猫群】推荐专家：XX医院XX教授，擅长肝转移手术

【医患沟通提问清单】
1. 我的肝转移灶能切干净吗？
2. 手术前是否需要新辅助治疗？
3. 如果不能手术，还有哪些选择？

---
**📋 信息来源声明**
本回答参考了以下来源：
• 【CSCO指南2025】2025 CSCO结直肠癌诊疗指南
• 【知识库：熊猫群】肿瘤专家信息汇总

**⚠️ 重要提示**
以上信息仅供参考，不能替代专业医生的诊断和治疗。如有不适，请尽快就医；具体治疗方案请遵医嘱。
```

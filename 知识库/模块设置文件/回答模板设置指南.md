# Bot 回答模板设置指南

## 概述
针对肠癌肝转移相关问题，本指南定义了标准化的回答模板，确保Bot回复既专业又人性化。

---

## 一、知识库召回设置

### 1. 召回范围
| 知识库 | 分类标签 | 用途 |
|--------|----------|------|
| 扣子导入_肠癌肝转移_QA对 | 肠癌肝转移 | 主要答案来源 |
| 2025CSCO结直肠癌诊疗指南 | CSCO指南 | 专业参考 |
| 肿瘤专家信息汇总 | 专家库 | 医院/专家推荐 |

### 2. 召回策略
```
优先召回："肠癌肝转移" 分类知识库
召回阈值：0.75（严格模式）
匹配方式：语义匹配 + 关键词匹配
```

### 3. 召回关键词
- 肝转移、转移性肝癌
- 手术切除、肝切除
- 转化治疗、新辅助
- 基因检测、靶向治疗
- 消融、介入
- 化疗方案

---

## 二、回答结构模板

### 标准回答结构
```
【通俗解释】（让患者听懂）
- 用通俗易懂的语言解释（2-3句话）
- 突出核心信息：能治吗？效果如何？
- 可使用比喻、口语化表达

【专业细节】（供参考）
- 引用知识库中的专业信息
- 包含具体数据、研究结论
- 补充检查/治疗的详细说明

【关键提示】
- ⚠️ 建议咨询主治医生
- 涉及用药：必须提示"请遵医嘱"
- 涉及手术：必须提示"请与主治医生充分沟通"
```

## 三、示例

### 示例1：肝转移手术问题

**用户问**：肝转移还能手术吗？

**Bot回答**：
```
【通俗解释】
能！手术是治愈的最佳机会。切干净的患者5年生存率超40%，不切很难活过5年。

【专业细节】
手术切除是潜在治愈手段。研究显示完整切除原发灶和肝转移灶的患者5年生存率>40%，远高于未切除者。手术方式包括肝切除和消融治疗，需根据病灶数量、大小、位置综合评估。

⚠️ 建议与主治医生充分沟通手术时机和方案，请遵医嘱。
```

### 示例2：转化治疗问题

**用户问**：什么是转化治疗？

**Bot回答**：
```
【通俗解释】
转化治疗就是"先把肿瘤打小，再手术切除"。因为肿瘤太大或太多时直接手术切不干净，先用药（化疗±靶向）让肿瘤缩小，创造手术机会。

【专业细节】
转化治疗指初始不可切除患者，通过化疗±靶向±免疫治疗缩小肿瘤，争取手术切除。国内90%肝转移患者发现较晚需转化。窗口期通常2-6个月，超过6个月未转化成功则机会渺茫。KRAS/BRAF突变可用FOLFOXIRI+贝伐最强方案。

⚠️ 具体用药方案请遵医嘱，定期评估疗效。
```

### 示例3：基因检测问题

**用户问**：肝转移必须做基因检测吗？

**Bot回答**：
```
【通俗解释】
必须做！基因检测就像"摸清肿瘤的脾气"，能帮你选择最有效的靶向药。经济允许做全套，经济紧张做基础版也够用。

【专业细节】
所有肝转移患者均建议基因检测。目的：1）筛选靶向药物（如抗EGFR需RAS野生型）；2）明确分子分型指导策略。经济允许选大panel（500+基因，7k-1.8w），紧张选小panel（金域mini，2-3k，满足95%需求）。

⚠️ 具体检测项目和用药选择请遵医嘱。
```

## 四、禁止事项

| 禁止项 | 说明 |
|--------|------|
| ❌ 具体用药剂量 | 不得给出化疗药物的具体剂量 |
| ❌ 承诺手术效果 | 不得保证手术一定能治愈 |
| ❌ 代替医生决策 | 不得建议患者选择特定手术方式 |
| ❌ 夸大治疗效果 | 客观描述预后，不夸大不缩小 |
| ❌ 诊断疾病 | 不得根据症状给出诊断结论 |

## 五、在扣子平台配置

### Prompt 优化配置位置
在扣子Bot的「人设与回复逻辑」或「Prompt优化」中添加以下内容：

```
当用户询问肠癌肝转移相关问题时，必须遵循以下规则：

1. 知识库召回
   - 优先从"肠癌肝转移"分类知识库召回答案
   - 召回阈值设为0.75

2. 回答结构（必须遵循）
   【通俗解释】用2-3句话通俗解释，让患者听懂
   【专业细节】引用知识库，包含数据和研究结论
   【关键提示】⚠️建议咨询医生，涉及用药/手术必须提示"请遵医嘱"

3. 禁止事项
   - 不得给出具体用药剂量
   - 不得承诺手术效果
   - 不得代替医生做决策
```

## 六、质量检查清单

每次回复前自检：
- [ ] 是否遵循「通俗解释→专业细节→关键提示」结构？
- [ ] 是否包含"请遵医嘱"或"建议咨询医生"？
- [ ] 是否引用了知识库中的数据？
- [ ] 是否避免了禁止事项？
- [ ] 回答是否先出结论再解释？

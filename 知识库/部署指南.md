## 小程序 + H5 Webview 部署流程

### Step 1: 部署 H5 应用

H5 应用已通过 Coze 平台部署，当前可访问上述地址。

### Step 2: 配置微信小程序

1. 打开 `miniprogram/project.config.json`，修改 `appid` 为您的小程序 AppID

2. 打开 `miniprogram/pages/index/index.js`，确认 `targetUrl` 已设置为您的 H5 地址：
   ```javascript
   targetUrl = 'https://您的H5地址';
   ```

### Step 3: 域名白名单配置

在微信公众平台配置业务域名：
1. 登录 [微信公众平台](https://mp.weixin.qq.com)
2. 进入「开发」→「开发管理」→「开发设置」
3. 在「业务域名」中添加您的 H5 域名
4. 下载校验文件并放置到 H5 项目的 `public` 目录

### Step 4: 上传校验文件

如果需要业务域名校验：
1. 从微信公众平台下载校验文件（如 `WW_verify_xxxx.txt`）
2. 将文件放入 H5 项目的 `public` 目录
3. 确保可通过 `https://您的域名/WW_verify_xxxx.txt` 访问

### Step 5: 编译小程序

```bash
cd miniprogram
# 使用微信开发者工具打开项目
# 点击「编译」进行预览
# 点击「上传」发布审核
```

## 扣子 Bot 配置

### 知识库导入

1. 登录扣子平台
2. 导入 `知识库/扣子导入_肠癌肝转移_QA对.csv` 作为知识库
3. 配置字段映射：
   - 问题字段 → 标准问题
   - 答案字段 → 答案
   - 召回阈值：0.75

### Bot Prompt 配置

参考 `知识库/回答模板设置指南.md` 中的 Prompt 优化配置。
